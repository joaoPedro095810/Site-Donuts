# 🍩 Donnuts Deliciosos

Site oficial da loja **Donnuts Deliciosos** — donuts artesanais feitos com amor.

## 📁 Estrutura do Projeto

```
donnuts-deliciosos/
├── index.html          → Página Inicial
├── sabores.html        → Página de Sabores
├── cardapio.html       → Cardápio completo
├── contato.html        → Fale Conosco
├── style.css           → Estilos globais
├── menu.js             → Menu responsivo
└── images/             → Imagens do site
    ├── donuts-hero.png
    ├── donut-morango.png
    ├── donut-chocolate.png
    ├── donut-tradicional.png
    ├── donut-morango-choco.png
    ├── donut-chocolate-morango.png
    ├── donut-pistache.png
    ├── donut-chocobranc.png
    └── donut-kinder.png
```

## 🚀 Como publicar no GitHub Pages

1. Crie um repositório no GitHub (ex: `donnuts-deliciosos`)
2. Faça upload de **todos os arquivos e a pasta `images/`**
3. Vá em **Settings → Pages → Source: main branch / root**
4. Aguarde alguns minutos — seu site estará em:
   `https://SEU_USUARIO.github.io/donnuts-deliciosos/`

> ⚠️ Importante: sempre suba a pasta `images/` junto com os HTMLs para as fotos aparecerem corretamente.

## 🎨 Personalização

Para atualizar contatos reais, edite `contato.html`:
- **E-mail**: substitua `donnutsdeliciosos@gmail.com`
- **WhatsApp**: substitua o número em `https://wa.me/55NUMERO`
- **Instagram**: substitua `@donnutsdeliciosos`

Para adicionar fotos reais de cada sabor, substitua os arquivos na pasta `images/` mantendo os mesmos nomes de arquivo.
